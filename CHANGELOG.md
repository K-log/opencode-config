# Changelog

## v0.1.0 — 2026-05-06

Initial release of the NBD opencode distribution.

### Added

- Modular plugin at `plugins/nbd/` with 26 modules
- 15 custom tools under `nbd_*` namespace
- Shared libs: `paths`, `storage`, `notifier`, `config`
- Vitest test suite (23+ tests)
- Custom theme `alpenglow.json` (contrast fixes)
- Custom agents: `build-parallelizer`, `code-reviewer`
- Custom commands including `interactive-review`
- Default config at `nbd.json`
- Skills: `building`, `code-quality`, `css`, `linting`, `planning`, `react`, `testing`, `typescript`
