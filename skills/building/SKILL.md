---
name: building
description: Best practices for building
---

When building out a plan, follow these core principles:

1. If the work is not broken up into concurrent tasks, use the `parallelize-task` agent (via the Task tool) to build a parallel work plan before proceeding.
2. Utilize the question tool for anything unclear or any unexpected situations. Do no deviate from the plan without confirmation.
3. Parallelize as many tasks as possible, utilize `build` agent (via the Task tool).
4. Always avoid unrelated changes. Stay focused on the current task and what the plan dictates.
