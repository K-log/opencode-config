## CSS

- Prefer CSS modules over inline styles.
- Prefer nested CSS selectors.
- Prefer CSS variables.
- Create new CSS variables for cases where multiple types of styling require the same value to remain in sync.
- Verify CSS featured used are supported by all major browsers.
