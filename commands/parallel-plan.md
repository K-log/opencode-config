Create a plan for parallel development.

The plan must include the following items:
- Tasks for each step of the plan. Make steps that can be worked on in parallel as much as possible.
- Include a directory structure diagram of all planned modifications to the project.
- Include sources for all references external to the project. Use [number] and a references footer of `[number]: <link>`.
- Use the _question_ tool for any clarification.

Minimize thinking and just propose the completed plan to the user.
